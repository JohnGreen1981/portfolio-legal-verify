# Legal Verify

AI-assisted legal answer verifier for Russian-law questions.

This is a sanitized portfolio repository. The original working folder is not copied here directly.

## What It Demonstrates

- Domain-specific AI workflow for legal answers.
- Generator + verifier architecture.
- Structured outputs for legal answer blocks and citation checks.
- Iterative correction loop: generate, verify, patch or regenerate.
- Hybrid validation: deterministic checks plus LLM-based verification.
